Cette partie permet de sélectionner des photographies selon le principe décrit dans la partie "Reconnaitre une lésion".

Une fois affichées, les photographies peuvent être placées dans la colonne de gauche qui représente le panier virtuel.
Il est possible de réaménager l'ordre des diapositives

Le panier ainsi constitué pourra Ítre enregistré pour une utilisation ultérieure comme montré plus bas.
 
 
L'ensemble des paniers que vous avez constitués se retrouvent dans une liste directement accessible depuis l'écran de démarrage de la partie formation.

Un clic sur le nom du panier déclenche sa projection plein écran.
