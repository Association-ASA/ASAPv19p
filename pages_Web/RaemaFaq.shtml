Recommandations découlant des retours d'information

Les difficultés signalées par les laboratoires lors de l'utilisation du site, pour saisir leurs résultats ou pour télécharger leurs rapports, nous conduisent à vous recommander les éléments suivants :

  • Eviter l'utilisation les versions antérieures à 9 de MicroSoft Internet Explorer™ comme navigateur Web et, par conséquent, l'utilisation de Windows XP™.
  • Activez Javascript afin de rendre la page plus réactive suite à la saisie de données non conformes.
  • Réglez, dans les options du navigateur Web, la modalité de téléchargement et de visualisation des documents pdf qui vous convient le mieux.
  • Installez le programme gratuit TeamViewer™ sur votre ordinateur si vous désirez un dépannage à distance par l'ASA.



Questions le plus souvent posées concernant le RAEMA


Question concernant les dates du RAEMA

Les dates d'envoi des colis, de dates limites pour les réponses concernant les RAEMA à venir ainsi que les dates de disponibilité des rapports et le lien pour les rapports généraux pour les RAEMA passés font périodiquement l'objet de questions de la part des laboratoires.

Réponse de l'ASA : La ligne "Agenda des RAEMA", dans l'onglet RAEMA du bandeau latéral du site, vous permet de connaître tous ces éléments.



Question concernant l'accès au site de l'ASA

Quelques laboratoires nous ont signalé que lors de l'accès au site (soit par l'adresse IP 217.128.177.16 soit par sa traduction littérale www.asa-spv.asso.fr) les liens ne fonctionnaient pas. Parfois l'écran était incomplet (manque du menu de navigation).
Dans tous les cas, cliquer sur un lien entraînait l'apparition d'une erreur (Service indisponible ).
Réponse de l'ASA : L'image qui apparaît sur l'écran est celle mise en mémoire par le navigateur. Vous pouvez donc réactualiser la page et si la manoeuvre est inopérante, vider le cache de votre navigateur.
NB : Les modalités de réalisation des manoeuvres proposées sont différentes selon les navigateurs et décrites dans leur mode d'emploi.



Précisions concernant le remplissage du formulaire

Le remplissage des données chiffrées ne doivent pas comporter d'unité. Si Javascript est activé sur votre ordinateur, il vous en informera tout de suite après la saisie. Dans le cas contraire, un écran apparaîtra après l'envoi du questionnaire.
C'est pourquoi nous vous conseillons de maintenir l'utilisation de Javascript dans votre navigateur.
Pour ce qui est des résultats chiffrés des germes, veillez à bien entrer 0.1 (zéro point un) et non 0,1 (zéro virgule un) qui est susceptible de provoquer une alerte.
NB : Les modalités d'activation désactivation de Javascript sont différentes selon les navigateurs et décrites dans leur mode d'emploi.



Question concernant l'envoi de l'accusé réception

L'envoi de l'accusé de réception n'est pas suivi, dans les minutes qui suivent de la réception à l'adresse souhaitée.
Réponse de l'ASA : Le mail est envoyé à l'adresse que vous indiquez : le premier réflexe est donc de vérifier la validité de cette adresse.
NB : L'adresse peut être enregistrée lors de votre envoi de façon à ce que le programme vous la propose par lors des prochaines visites.
Cette adresse peut être modifiée sans inconvénient.



Question concernant la récupération des rapports

Quelques laboratoires nous ont signalé que, lors de la récupération des rapports, les liens ne fonctionnaient pas.
Réponse de l'ASA : Les navigateurs Web ont récemment changé le fonctionnement du téléchargement. Fréquemment, ils enregistrent sans l'indiquer à l'utilisateur le fichier qui se trouve sous le lien de la page.
Il s'en suit une impression de non-téléchargement.
Pour visualiser où le navigateur enregistre le rapport de campagne désigné, faîtes un clic droit sur le lien et demandez le menu local qui propose d'enregistrer la cible du lien sur votre disque dur. Dans le dialogue que vous proposera alors le navigateur, choisissez un emplacement où vous retrouverez facilement le fichier souhaité.
NB : Les modalités de téléchargement sont différentes et modifiables par l'utilisateur selon les types et versions des navigateurs. Elles sont décrites dans les modes d'emploi.
