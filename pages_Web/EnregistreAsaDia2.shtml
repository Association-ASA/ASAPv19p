Enregistrement d'AsaDia 2


L'enregistrement d'AsaDia 2 est effectué après avoir préalablement obtenu une licence d'utilisation auprès de l'ASA sous forme d'une adresse mail.
Une fois cette adresse mail enregistrée, veuillez la saisir dans le premier champ et remplir les autres champs proposés.
Cliquer sur le bouton "Envoyer" pour pouvoir recevoir le code permettant d'utiliser AsaDia 2
