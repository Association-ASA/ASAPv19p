EXEMPLE

Entrez dans le tableau vos exigences en termes de NQA, NQL, risques fournisseur et client, le tableau vous renvoie le plan n1 = 45, c1 = 0, c1' = 2, n2 = 308, c2 = 6.

Si après prélèvement d'un premier échantillon d'effectif 45, vous n'obtenez aucune unité non conforme, vous accepterez le lot, si vous obtenez plus de 2 unités non conformes, vous le rejetterez et si vous en avez 1 ou 2, vous prélèverez un deuxième échantillon d'effectif 308. A l'issu de ce contrôle, votre lot sera accepté si vous obtenez moins de 6 unités non conformes dans vos 2 échantillons.
